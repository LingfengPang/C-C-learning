## 哈希表题目
141. 环形链表
判断是否有环
思路，把遍历过的放到set里面
```cpp
// N N
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_set<int> s;
        for (int x: nums) {
            if (s.find(x) != s.end()) {
                return true;
            }
            s.insert(x);
        }
        return false;
    }
};

```
